# Improvements-to-the-Milan-language-compiler
Лабораторная работа №4 «Доработка компилятора языка Milan» по дисциплине «Математическая логика и теория формальных языков»

Модифицировать компилятор языка Милан (CMILAN) путём добавления новой
управляющей конструкции – цикла с параметром for. Реализовать оператор цикла,
а также проверить программу на для нового оператора.

for <переменная> from <выражение> to <выражение>
do <список операторов> od

Задачи.
1. Анализ цикла for в различных языках программирования.
2. Реализация оператора цикла for в компиляторе языка Milan в соответствии с
заданным синтаксисом.
3. Генерация объектного кода для виртуальной машины.
4. Проверка программы на тестовых примерах
